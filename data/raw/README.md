# Raw Data Directory

Place your raw input documents here.

Supported formats:
- PDF files (`.pdf`)
- Text files (`.txt`)
- Markdown files (`.md`)
- Word documents (`.docx`)
- HTML files (`.html`)

Example structure:
```
raw/
├── research_papers/
│   ├── paper1.pdf
│   └── paper2.pdf
├── documentation/
│   └── manual.txt
└── articles/
    └── article.md
```
